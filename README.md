# 紙 から PDF を作るやつ
裏表の用紙に書かれた書類を紙送りのついたスキャナで取り込んで PDF にするためのスクリプトです. 
裏表を同時に取り込んでくれないので表裏で別々にpdfを用意してから交互にマージしています.
詳細は [PyPDF2 の Documentation](https://pythonhosted.org/PyPDF2/)を読んでください.
